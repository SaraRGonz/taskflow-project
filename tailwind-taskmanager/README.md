# Task Manager - Tailwind CSS Edition

Una evolución de la aplicación de gestión de tareas, ahora reconstruida utilizando **Tailwind CSS**. Este proyecto combina la lógica robusta de JavaScript Vanilla con la flexibilidad y rapidez de diseño de un framework de utilidades CSS moderno.

---

## Características Principales

*   **Estilizado con Tailwind CSS:** Diseño moderno, limpio y totalmente responsivo (adaptable a móviles y escritorio) utilizando clases de utilidad.
*   **Persistencia de Datos:** Las tareas se almacenan en el `localStorage` del navegador, asegurando que no se pierdan al recargar la página.
*   **Gestión de Prioridades:** Sistema visual para clasificar tareas por importancia (Alta, Media, Baja) con indicadores de color.
*   **Manipulación Dinámica del DOM:** Interfaz interactiva y rápida sin necesidad de recargas.

## Tecnologías Utilizadas

*   **HTML5 Semántico:** Estructura base de la aplicación.
*   **Tailwind CSS:** Framework CSS "utility-first" para el diseño visual, tipografía, espaciado y colores.
*   **JavaScript (ES6+):** Lógica de la aplicación, manejo de eventos y persistencia de datos.
*   **Node.js & NPM:** Para la gestión de dependencias y el proceso de construcción (build) de Tailwind.
